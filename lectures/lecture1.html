Сортировка вставками (insertion sort)

Устойчивая сортировка, эффективная на небольших объемах данных.

На каждом шаге элемент перемещается назад по массиву до тех пор, пока перед ним находится больший элемент.

def insertion_search(a):
# 
    for i in range(1, len(a)):
        key = a[i]
        for k in range(i - 1, -1, -1):
            if key < a[k]:
                a[k], a[k + 1] = a[k + 1], a[k]
            else:
                break
    return a
#

«Данный алгоритм можно ускорить при помощи использования бинарного поиска для нахождения места текущему элементу в отсортированной части. Проблема с долгим сдвигом массива вправо решается при помощи смены указателей» — Википедия
